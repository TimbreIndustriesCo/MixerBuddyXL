# MixerBuddyXL
Mixer Buddy XL is a 45HP MIDI controller in Eurorack format. It is designed to be the partner in crime of Eurorack modules with a USB host function, or a PC. Examples of these are the 1010 Music Bluebox, Expert Sleepers FH-2 and the Westlicht Per|Former. It’s a perfect companion to control a mixer.
